# Security Policy

## Supported Versions

Currently supported security updates for version 0.0.0.

## Reporting a Vulnerability

If you discover a security vulnerability, please email it to security@[your-domain].com with:

1. Description of the vulnerability
2. Steps to reproduce
3. Potential impact
4. Suggested fix (if known)

**Response Time**: We aim to respond within 48 hours.

**Do NOT**:
- Open public issues about security vulnerabilities
- Discuss vulnerabilities in public channels

## Security Features

- Client-side only application (no server-side code execution)
- No storage of sensitive data
- TypeScript strict mode enabled
- Automated dependency scanning via GitHub Actions
- Security headers configured via Netlify

## Dependency Management

- Weekly automated dependency updates via Dependabot
- Security audits run on every pull request
- Critical vulnerabilities patched within 7 days
